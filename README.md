## Run Locally

Clone the project

```bash
  git clone https://github.com/alexhp123/d--os
```

Go to the project directory

```bash
  cd d--os
```

Install vscode

```bash
sudo pacman -S code
```

Start the server

extensions > Live Preview > right click on index.html > show Preview
